# Bringing Blockchain Intelligence to Your Workflows: Introducing the n8n Chainlink Node

We're excited to announce the release of our latest community contribution: the n8n-nodes-chainlink package! Developed by Velocity BPA, this powerful integration brings Chainlink's decentralized oracle services directly into your n8n automation workflows.

## Why Chainlink Integration Matters

As blockchain technology becomes increasingly integral to modern business operations, the need to bridge on-chain and off-chain data has never been greater. Chainlink is the industry-standard decentralized oracle network that enables smart contracts to securely interact with real-world data, events, and payments.

Until now, integrating Chainlink's services into automation workflows required custom development and deep blockchain expertise. Our new n8n node changes that entirely, making enterprise-grade blockchain functionality accessible through n8n's intuitive visual interface.

## Key Features

The n8n-nodes-chainlink package provides seamless access to Chainlink's core services:

**Price Feeds**: Pull real-time price data for cryptocurrencies, commodities, and traditional assets directly into your workflows. Perfect for financial reporting, trading alerts, or DeFi application monitoring.

**VRF (Verifiable Random Function)**: Generate provably fair random numbers for gaming applications, NFT distributions, or any use case requiring cryptographically secure randomness.

**Automation**: Integrate with Chainlink Automation (formerly Keepers) to trigger workflows based on on-chain conditions or time-based events.

**CCIP (Cross-Chain Interoperability Protocol)**: Enable cross-chain messaging and token transfers, allowing your workflows to interact with multiple blockchain networks seamlessly.

## Getting Started

Installation is straightforward. Simply run:


npm install n8n-nodes-chainlink


Once installed, the Chainlink node will appear in your n8n node palette, ready to drag and drop into your workflows. The node includes comprehensive configuration options and supports all major Chainlink services across multiple blockchain networks.
